# 🏪 برنامج أبو محمود السوري للمحاسبة

برنامج محاسبة متكامل مستوحى من برنامج الأمين للمحاسبة، يعمل مباشرة في المتصفح دون الحاجة لتثبيت أي برنامج.

## المميزات

- **إدارة الزبائن**: إضافة وتعديل وحذف الزبائن مع حفظ بياناتهم
- **نظام الفواتير الكامل**: إنشاء فواتير بيع وشراء ومرتجع
- **جدول الأصناف**: إدخال النوع، الوصف، الوحدة، الكمية، السعر، الخصم، والمجموع
- **حساب تلقائي**: حساب المجاميع والخصومات والضرائب تلقائياً
- **حفظ الفواتير**: حفظ الفواتير محلياً في المتصفح
- **طباعة الفواتير**: طباعة فواتير احترافية
- **تصدير البيانات**: تصدير الفواتير بصيغة CSV
- **التقارير**: عرض إحصائيات المبيعات وأفضل الزبائن

## اختصارات لوحة المفاتيح

| الاختصار | الوظيفة |
|----------|---------|
| F2 | إضافة سطر جديد |
| F5 | فاتورة جديدة |
| F9 | حفظ الفاتورة |
| F8 | طباعة الفاتورة |
| Ctrl+Delete | حذف السطر المحدد |

## كيفية الاستخدام

1. افتح ملف `index.html` في أي متصفح
2. اختر اسم الزبون من القائمة الجانبية أو أضف زبوناً جديداً
3. أدخل الأصناف في الجدول (النوع، الكمية، السعر)
4. يتم حساب المجاميع تلقائياً
5. اضغط "حفظ الفاتورة" أو F9 لحفظها
6. اضغط "طباعة" أو F8 لطباعتها

## التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript (Vanilla)
- localStorage للحفظ المحلي
