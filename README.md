# 🏪 برنامج أبو محمود السوري للمحاسبة - الإصدار 2.0

برنامج محاسبة متكامل متوافق مع الجوال، يعمل مباشرة في المتصفح مع إمكانية ربط البيانات بحساب Google الخاص بك عبر Firebase.

## ✨ المميزات الجديدة في الإصدار 2.0

- **واجهة متوافقة مع الجوال**: تصميم متجاوب يعمل بسلاسة على جميع الأجهزة
- **بيانات فارغة افتراضياً**: ابدأ من الصفر وأضف زبائنك الخاص بك
- **أرقام نظيفة**: إزالة الأصفار الزائدة من المجاميع
- **ربط Firebase**: حفظ البيانات تلقائياً في السحابة ومزامنتها عبر أجهزتك
- **تسجيل الدخول بـ Google**: حماية بياناتك برابط آمن مع حسابك

## 🚀 كيفية البدء

### 1. التشغيل المحلي (بدون Firebase)
1. فك ضغط الملف
2. افتح `index.html` في أي متصفح
3. ابدأ بإضافة زبائنك والفواتير

البيانات ستُحفظ محلياً في جهازك فقط.

### 2. ربط مع Google (Firebase)

#### الخطوة 1: إنشاء مشروع Firebase
1. اذهب إلى [Firebase Console](https://console.firebase.google.com/)
2. انقر على "إنشاء مشروع جديد"
3. أدخل اسم المشروع (مثل: "Abu-Mahmoud-Accounting")
4. اتبع الخطوات

#### الخطوة 2: تفعيل المصادقة
1. في Firebase Console، اذهب إلى **Authentication**
2. انقر على **Get Started**
3. فعّل **Google** كطريقة تسجيل دخول
4. أضف بريدك الإلكتروني كمستخدم اختبار

#### الخطوة 3: إنشاء Firestore Database
1. اذهب إلى **Firestore Database**
2. انقر على **Create Database**
3. اختر **Start in test mode** (للاختبار)
4. اختر المنطقة الجغرافية

#### الخطوة 4: الحصول على بيانات المشروع
1. اذهب إلى **Project Settings** (رمز الترس)
2. انسخ بيانات المشروع:
   - `apiKey`
   - `authDomain`
   - `projectId`
   - `storageBucket`
   - `messagingSenderId`
   - `appId`

#### الخطوة 5: تحديث الكود
افتح ملف `app.js` وابحث عن:
```javascript
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_PROJECT.appspot.com",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID"
};
```

استبدل القيم بالبيانات التي نسختها من Firebase.

#### الخطوة 6: تفعيل المزامنة
- افتح البرنامج في المتصفح
- انقر على ☰ (القائمة)
- اختر "تسجيل الدخول (Google)"
- سجّل دخولك ببريدك الإلكتروني
- ستظهر رسالة "☁️ متزامن" في الأسفل

## 📋 كيفية الاستخدام

### إضافة زبون جديد
1. انقر على زر "اختر" بجانب اسم الزبون
2. انقر على "+ إضافة زبون جديد"
3. أدخل البيانات وانقر "حفظ"

### إنشاء فاتورة
1. اختر الزبون
2. أضف الأصناف (النوع، الكمية، السعر)
3. المجموع يُحسب تلقائياً
4. انقر "حفظ" أو اضغط F9

### طباعة الفاتورة
- انقر "طباعة" أو اضغط F8

### عرض التقارير
- انقر على ☰ ثم "التقارير"

## ⌨️ اختصارات لوحة المفاتيح

| الاختصار | الوظيفة |
|----------|---------|
| F2 | إضافة صنف جديد |
| F5 | فاتورة جديدة |
| F9 | حفظ الفاتورة |
| F8 | طباعة الفاتورة |

## 🔒 الأمان والخصوصية

- البيانات تُحفظ محلياً في جهازك
- عند تفعيل Firebase، تُحفظ البيانات في السحابة بشكل آمن
- فقط أنت يمكنك الوصول لبياناتك
- Firebase يستخدم تشفير من طرف إلى طرف

- **Storage**: localStorage (محلي) + Firebase Firestore (سحابة)
- **Authentication**: Firebase Authentication with Google
- **Responsive**: تصميم متجاوب يتكيف مع جميع الأحجام
